    Download IMTBX+Grppr¶

    Precompiled binaries for Win-x64 are available in the Releases section of the GitHub repository.

    What's in the distribution¶

    • IMTBX.exe - The peak detection program for Waters .RAW files.
    • IMTBX-View.exe - Data viewer for both regular MS and IM-MS data from Waters' instruments.
    • grppr-0.3.6.jar - Isotopic clustering program.

    Requirements¶

    • IMTBX.exe and IMTBX-View.exe
    • Require .NET 4.5.2 installed. If you're on Windows 10, you already have it. Most likely you have it on older systems as well, but if the program doesn't run, you can get it here.
    • grppr-0.3.6.jar
    • Requires Java 1.8 installed. If you don't want to install the standard one from Oracle, you can get an alternative OpenJDK pre-built distribution for Win/Linux/Max here.
